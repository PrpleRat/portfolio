# Panium

**Handpan tactile — iOS audio**

| | |
|---|---|
| Stack | Swift · SwiftUI · AVAudioPlayer |
| Approche | 100 % offline · samples WAV |
| Bundle ID | `com.prple.panium` |

## Fonctionnalités

- 7 gammes, reverb multi-niveaux, accord 440 / 432 Hz
- Layout centre + anneau, labels notes
- Vélocité selon position du toucher
- Polyphonie jusqu’à 32 notes

## Pourquoi c’est intéressant

- Audio temps réel / UX tactile
- Port soigné d’une app musicale en SwiftUI natif
