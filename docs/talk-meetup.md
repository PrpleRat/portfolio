# Talk meetup — pitch prêt

**Titre :** TestFlight depuis Windows — CI iOS sans Mac  
**Durée :** 10–12 min + 5 min questions  
**Niveau :** junior → confirmé mobile  
**Cible idéale :** [MUG Toulouse](https://www.meetup.com/mug-toulouse-mobile-user-group/) · La Mêlée / Cantine · meetup open source Occitanie  

## Pitch (à coller aux orga)

```text
Salut !

Je suis Enzo, développeur iOS / mobile à Toulouse.
J’ai open-sourcé mon pipeline GitHub Actions qui archive et envoie sur TestFlight
sans Mac local (XcodeGen / Expo prebuild → IPA → ASC).

Repo : https://github.com/enzo-jouet/ios-testflight-ci

Est-ce que vous auriez un slot de ~10 min sur un prochain meetup
pour un retour d’expérience concret (secrets, certificat réutilisé, pièges ITMS) ?

Dispo soirs de semaine. Merci !
Enzo — 07 68 20 98 54
```

## Plan des 12 minutes

1. **Le problème** (1 min) — pas de Mac, besoin de shipper quand même  
2. **L’archi** (3 min) — schéma Actions → archive → TestFlight  
3. **Les 3 pièges** (4 min) — cert unique, CFBundleVersion, signature ITMS  
4. **Démo live** (3 min) — ouvrir le repo + workflow + (idéal) replay d’un run  
5. **Takeaways** (1 min) — lien repo + Q&A
