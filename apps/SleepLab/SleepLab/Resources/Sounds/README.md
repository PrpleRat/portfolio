# Sons réveil SleepLab

Fichiers intégrés dans le bundle (générés Xcode depuis `SleepLab/Resources/Sounds/`).

| Fichier | Usage |
|---------|--------|
| `rain.mp3` | Pluie — boucle réveil |
| `bowl.mp3` | Bol tibétain |
| `birds.mp3` | Oiseaux du matin |
| `ocean.mp3` | Vagues |
| `gentle_piano.mp3` | Harpe douce |
| `*.wav` | Notifications locales (≤ 8 s) |
